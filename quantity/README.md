### Разработка библиотеки для работы с физическими величинами в C++.

*C++ позволяет отловить многие ошибки, связанные с типами данных, еще на этапе компиляции. Однако встроенной системы числовых типов часто бывает недостаточно для детектирования ошибок, связанных с различными вычислениями.*

*В частности, при работе с физическими величинами, важно соблюдать не только корректность математических операций, но и следить за соответствием размерностей величин. Например, нельзя складывать массу и длину, а при умножении скорости на время должна получиться длина.*

*Создание библиотеки, которая позволит проводить вычисления с физическими величинами с проверкой размерностей на этапе компиляции, поможет избежать непреднамеренных ошибок и упростит разработку физических моделей.*

#### Требуется реализовать

1. **Шаблон структуры `Quantity`**

   - Шаблон структуры `Quantity` с числовыми шаблонными параметрами `T`, `L`, `M`, представляющими показатели степеней времени, длины и массы соответственно. Будем считать, что показатели степеней времени, длины и массы могут быть только целыми числами.
   - Поле `value` типа `double`, содержащее численное значение величины.

2. **Переопределение операций**

   - **Операции сложения и вычитания**

     - Операции `+` и `-` для величин с одинаковыми размерностями.
     - Операции `+=` и `-=` для величин с одинаковыми размерностями.

   - **Унарный плюс/минус**
   
   - **Операции умножения и деления**

     - Операции `*` и `/` для операций между величинами, где показатели размерностей складываются и вычитаются соответственно.
     - Операции `*` и `/` для умножения и деления величин на скаляр (`double`).
     - Операции `*=` и `/=` для умножения и деления величин на скаляр с присваиванием.

   - **Операции сравнения**

3. **Псевдонимы типов для основных физических величин**

   - Определить псевдонимы типов для часто используемых физических величин, таких как:
     - `Time` (время)
     - `Length` (длина)
     - `Mass` (масса)
     - `Speed` (скорость)
     - `Acceleration` (ускорение)
     - `Force` (сила)
     - `Energy` (энергия)

#### Примеры использования

Больше примеров можно найти в тестах.

**Пример 1: Вычисление скорости**

```cpp
Length distance{100.0};   // 100 метров
Time time{9.58};          // 9.58 секунд

const auto speed = distance / time;  // speed имеет тип Speed
```

**Пример 2: Ошибка при несовместимых операциях**

```cpp
Length l{5.0};
Mass m{2.0};

// Следующая строка вызовет ошибку компиляции,
// так как нельзя складывать длину и массу
const auto invalid = l + m;
```

#### Дополнительное задание

Поддержите создание величин с помощью литералов. Например:

```cpp
auto l = 5.0_m;    // 5 метров (Length)
auto t = 9.58_s;   // 9.58 секунд (Time)
auto m = 100.0_kg; // 100 килограмм (Mass)
auto s = 10.0_mps; // 10 метров в секунду (Speed)
auto a = 9.8_mps2; // 9.8 метров в секунду в квадрате (Acceleration)
auto f = 10.0_N;   // 10 Ньютон (Force)
auto e = 100.0_J;  // 100 Джоулей (Energy)
```
