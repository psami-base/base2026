### Hangman: Word Generator

В этой подзадаче необходимо реализовать класс `WordGenerator`, отвечающий за загрузку слов и выдачу случайного слова для новой игры.

##### Идея

`WordGenerator` хранит словарь в памяти и генератор псевдослучайных чисел.

После загрузки слов из файла класс должен уметь возвращать случайное слово с равномерным распределением по индексам.

##### Ожидаемый интерфейс

* Конструктор:
  * принимает путь к файлу слов;
  * принимает seed генератора случайных чисел;
  * загружает слова из файла.

* `LoadWordsFromFile(path)`
  * перечитывает словарь из файла;
  * оставляет только валидные слова;
  * переводит слова в нижний регистр;
  * при ошибках бросает исключение.

* `GetRandomWord()`
  * возвращает случайное слово из загруженного словаря;
  * если словарь пуст, бросает исключение.

##### Требования к данным

1. Слова читаются из текстового файла, разделители — пробелы/переводы строк.
2. В словарь попадают только слова, состоящие из букв.
3. После загрузки словарь не должен быть пустым.

##### Генерация случайных чисел в C++

Используйте современные средства `<random>`:

* генератор (`std::mt19937_64`);
* распределение (`std::uniform_int_distribution<size_t>`).

Не используйте `std::rand`/`std::srand`.

Подробное объяснение и рекомендации по генерации случайных чисел смотрите в задаче `random`:

* `../../random/README.md`

##### Замечания

1. `LoadWordsFromFile` желательно реализовать с сильной гарантией безопасности исключений (при ошибке старое состояние сохраняется).
2. Преобразование символов через `std::tolower`/`std::isalpha` делайте с приведением к `unsigned char`.
3. В качестве решения ожидаются `word_generator.hpp` и `word_generator.cpp`.
